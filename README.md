## 🚀 Возможности

🛠️ Настройка производительности и визуальных эффектов

🔧 Управление автозапуском и системными сервисами

🎨 Современный и аккуратный дизайн интерфейса

⚡ Быстрая и безопасная оптимизация системы

## 💻 Системные требования

Windows 7/8.1/10/11

.NET 8.0 или выше

## 📥 Установка

1. [Скачайте последнюю версию](https://github.com/vitalikkontr/WindowsTweaks-Pro/releases/latest)
2. Запустите установщик WindowsTweaks-Professional-Setup-v3.0.1.exe
3. Следуйте инструкциям мастера установки


## 📝 Использование

Запустите программу от имени администратора

Выберите категорию твиков для настройки

Нажмите «Применить выбранные твики»

Дождитесь завершения процесса

## 🛠️ Технологии

Язык: C#

Фреймворк: .NET 8.0

UI: Windows Forms

IDE: Visual Studio

## Сборка проекта

```bash
git clone https://github.com/vitalikkontr/WindowsTweaks-Pro.git
cd WindowsTweaks-Pro
dotnet build
```
## ⚠️ Безопасность

Работает только с настройками Windows, не удаляет важные файлы

Не требует права администратора для применения твиков
